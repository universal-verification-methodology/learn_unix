# Module 03 examples — Absolute vs relative paths

Track A (real Unix). Each folder was adapted from `learn_unix_git`.

## `paths/`

See [`examples/paths/README.md`](examples/paths/README.md) if present.

**Try these** (from `examples/paths`):

```bash
# pwd — absolute path of “here”
pwd

# ls -la — what relative names refer to
ls -la

# cd .. — relative: up one level
cd ..
pwd

# cd paths — relative: back into this folder
cd paths

# cd ~ — home (tilde)
cd ~
pwd
```
